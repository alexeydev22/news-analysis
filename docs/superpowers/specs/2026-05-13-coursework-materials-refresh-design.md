## Актуальное состояние проекта, которое нужно отразить

Проект представляет собой диалоговую систему для анализа экономических новостей.

Ключевые возможности:

- загрузка и индексирование датасета FNSPID;
- работа с выборкой 50 000 экономических новостей;
- классификация влияния новости по классам `positive`, `neutral`, `negative`;
- сравнение трех моделей:
  - TF-IDF + Logistic Regression;
  - embedding-logreg;
  - tiny-transformer-classifier;
- формирование ML-отчета с метриками;
- поиск релевантных новостей через векторное хранилище;
- RAG-сценарий ответа на вопрос пользователя;
- подключение языковой модели через внешний API для аналитического прогноза;
- тематический прогноз по группам новостей;
- пользовательский интерфейс на React;
- микросервисная backend-архитектура.

В материалах нельзя представлять проект как простой чат-бот. Его нужно описывать как автоматическую диалоговую аналитическую систему, объединяющую ML-классификацию, retrieval, RAG и внешний LLM API.
